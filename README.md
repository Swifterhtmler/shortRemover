# ShortRemover

ShortRemover is a safari extension for ios,ipadOS and MacOS via (macOS catalyst). 




## Installation

You can download the app from app store


## FAQ

#### Is the app free ?

The app is 100% free

#### Does it work on ios on safari ?

The app works on safari on ios.

#### DOes the app work in chrome browser ?

The app does not work on chrome as its desgined for safari.



## Features

- Shorts visible/hidden toggle
- No data collection
- Works on mobile and desktop
