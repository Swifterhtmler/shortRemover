# ShortRemover

ShortRemover is a Safari extension that removes or hides short-form video sections (such as “Shorts”) to create a more focused browsing experience.

Built for iOS, iPadOS, and macOS using Mac Catalyst, with an emphasis on simplicity, performance, and privacy.

The app reached over 500 users within the first two days of launch, with a 39.8% conversion rate. Growth was entirely organic, with no marketing.

---

## Download

Available on the [App Store](https://apps.apple.com/fi/app/shortremover/id6761337873?l=fi)

## Features

* Toggle visibility of short-form content
* Works across iPhone, iPad, and Mac
* No data collection or tracking
* Lightweight and efficient

---

## Technical Overview

* Built using Safari Web Extensions
* Shared codebase across iOS, iPadOS, and macOS via Mac Catalyst
* Minimal DOM manipulation for performance and reliability
* Native integration with Safari extension settings

## Design

ShortRemover is intentionally simple:
